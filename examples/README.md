# Examples

These example projects are checked in to show what real agent outputs and tournament state look like.

Included examples:

- [`pilot-scripted/`](pilot-scripted)
  Small scripted pilot with seed agents and their generated strategies/results.
- [`trustworthy-pilot/`](trustworthy-pilot)
  Compact ensemble-focused pilot showing evolved agents, predictions, and shared knowledge.

Intentionally not included:

- large `live-*` run directories
- transient local state
- CLI auth/config files

Use these directories as reference datasets, not as the default writable workspace for new runs.
